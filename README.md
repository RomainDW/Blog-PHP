## Installation
* Clone : 
```bash
git clone https://github.com/RomainDW/Blog-PHP.git
```
* Update composer :
``` bash
composer update
```

* Create Database
* Enter your database configuration in `/Engine/Config.php`
* Import in your database `database.sql`
